Principais tags html e seletores css

Tags basicas do HTML:

A tag HTML é usada para inicio e fim de um documento em html, HTML é uma marca que define um conteúdo de documento das respectivas linguagem, Ela envolve textos e outros elementos.

A tag BODY

A tag BODY define um corpo de um site, È na tag BODY que se encotra as informaçoes visiveis para o usuário.

Tag < p >

A tag < p > é usada para indicar um paragrafo
